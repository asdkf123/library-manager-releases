# Library Manager v3.2.0 릴리즈 노트

## 🆕 새로운 기능

### 🗂️ 라이브러리 백업 및 복구 (Beta)
- **소중한 데이터를 안전하게**: 이제 라이브러리의 모든 메타데이터를 파일 하나로 백업할 수 있습니다.
- **완벽한 복원력**:
    - **포함 항목**: 읽기 기록, 별점, 시리즈 분류, 작가 태그, 직접 설정한 표지 이미지 등.
    - **제외 항목**: 저작권 보호를 위해 실제 만화/소설 파일은 백업되지 않습니다. (파일 경로는 유지됨)
- **간편한 이동**: 컴퓨터를 포맷하거나 다른 PC로 이동할 때, 백업 파일(.zip) 하나만 있으면 기존 라이브러리 환경을 그대로 복구할 수 있습니다.
- **설정 메뉴**에서 '백업 생성' 및 '복구' 기능을 바로 이용해 보세요.

---
*이번 업데이트는 데이터 안정성을 위한 백업 기능 추가에 집중했습니다.*
